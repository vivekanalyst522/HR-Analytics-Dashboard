# HR-Analytics-Dashboard

📊 HR Analytics Dashboard

This project is an interactive HR Analytics Dashboard built to analyze employee attrition, workforce demographics, salary distribution, and job role performance. The dashboard provides valuable insights into employee turnover trends and helps organizations make informed HR decisions.

🔹 Features:
• Employee Attrition Analysis
• Gender-wise Attrition Comparison
• Education-wise Attrition Distribution
• Age Group Analysis
• Salary Slab Analysis
• Job Role Attrition Tracking
• Years at Company Analysis
• Interactive Filters and KPIs

🛠 Tools Used:
• Microsoft Excel
• Pivot Tables
• Pivot Charts
• Data Visualization

📈 Business Benefits:
• Identify key factors influencing employee attrition.
• Improve employee retention strategies.
• Monitor workforce demographics and trends.
• Support data-driven HR decision-making.
• Enhance organizational workforce planning.
